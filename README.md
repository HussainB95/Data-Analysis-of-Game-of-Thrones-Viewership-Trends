# Exploratory-Data-Analysis-of-Game-of-Thrones-Viewership-Trends

## Project Overview
This project analyzes Game of Thrones episode viewership trends using Python.

## Objectives
- Analyze audience growth
- Compare season popularity
- Evaluate director performance
- Identify top-performing episodes

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Power BI

## Key Insights
- Season 7 had highest audience
- Finale episodes gained highest viewership
- Viewership grew steadily over time
